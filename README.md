<h2 align="center"><br>Magnifier</h2>
<p align="center">Smooth magnifier utility for Windows similar to the one built into macOS.</p>

<video src='https://i.imgur.com/7KFp7J9.mp4' width=180/>

## Usage: `Win+Shift` +

- Scrollwheel to zoom in/out
- Q to quit
